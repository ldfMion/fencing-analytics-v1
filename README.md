# Fencing Analysis

This project analyzes fencing data from a CSV file and calculates various metrics.

## Project Structure

- `main.py`: The entry point of the application.
- `csv_data_provider.py`: A data provider that reads from a CSV file.
- `data_provider.py`: An abstract base class for data providers.
- `fencing_analysis.py`: The main analysis class.
- `metrics.py`: Contains the logic for calculating metrics.
- `process_df.py`: Contains functions to process the dataframes.
- `Elite Invitationals 2025 Analytics.csv`: The data file.

## How to run

1. Install the dependencies:
```bash
pip install -r requirements.txt
```
2. Run the main script:
```bash
python3 main.py
```
